# Data_Analyst_Portfolio

| Project | Description | Tools | Impact / Value |
| - | - | - | - |
| **Regional Sales Optimization** | Analyzed historical sales data to identify trends by region and product, generating actionable insights for the commercial team. | SQL, Power BI | Enables strategic decision-making for sales and resource allocation. |
| **Customer Data Cleaning and Transformation** | Prepared a messy dataset by removing duplicates, filling missing values, and standardizing formats for reliable analysis. | Python (pandas), SQL | Improves data quality and reduces errors in reports and later analyses. |
| **Customer Behavior Segmentation** | Identified customer groups based on purchase frequency, spending, and preferences, supporting personalized marketing strategies. | SQL, Power BI | Helps target more effective marketing campaigns and increase retention. |
| **Performance KPI Dashboard** | Built an interactive dashboard with key metrics such as revenue, churn, and sales by product, updated periodically. | Power BI, Excel | Facilitates fast, data-driven decision-making for management. |
| **Monthly Sales Forecasting** | Analyzed historical trends and created simple sales forecasts per product using moving averages and basic linear regression. | Excel, Python | Enables proactive planning of inventory and sales targets. |
