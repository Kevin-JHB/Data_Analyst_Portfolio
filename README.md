# Data_Analyst_Portfolio

|Proyecto | Descripción	| Herramientas	| Impacto / Valor|
|-|-|-|-|
Optimización de Ventas Regionales	| Analicé datos de ventas históricas para identificar tendencias por región y producto, generando insights accionables para el equipo comercial.	| SQL, Power BI	| Permite tomar decisiones estratégicas de ventas y asignación de recursos.
Limpieza y Transformación de Datos de Clientes	| Preparé un dataset desordenado eliminando duplicados, rellenando valores faltantes y estandarizando formatos para análisis confiable.	| Python (pandas), SQL	| Mejora la calidad de los datos y reduce errores en reportes y análisis posteriores.
Segmentación de Clientes por Comportamiento	| Identifiqué grupos de clientes según frecuencia de compra, monto y preferencias, facilitando estrategias de marketing personalizadas.	| SQL, Power BI	| Ayuda a dirigir campañas de marketing más efectivas y aumentar retención.
Dashboard de KPIs de Desempeño	| Creé un panel interactivo con métricas clave como ingresos, churn y ventas por producto, actualizado periódicamente.	| Power BI, Excel	| Facilita la toma de decisiones rápidas y basadas en datos para la gerencia.
Proyección de Ventas Mensuales	| Analicé tendencias históricas y calculé pronósticos simples de ventas por producto usando promedio móvil y regresión lineal básica.	| Excel, Python	| Permite planificar inventario y objetivos de ventas con anticipación.
